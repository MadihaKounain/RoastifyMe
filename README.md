RoastifyMe is a fun AI-powered Python application that analyzes user input and generates witty roasts or positive compliments.
The project combines rule-based intelligence with creative text generation to deliver humorous, engaging, and interactive responses.

Designed as a lightweight AI-style application, RoastifyMe demonstrates how structured data, text processing, and modular Python architecture can be used to build entertaining intelligent systems.
